## 🚀 About

> **Automated insight generation saving analysts 3+ hours per dataset review cycle**

**E-Commerce EDA Tool** is a production-ready **Data Science** project built with **Python, Pandas, Seaborn, ReportLab**.
Deployed on **Render** with full CI/CD pipeline.

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/alam025/eda-tool.git
cd eda-tool

# 2. Install
pip install -r requirements.txt

# 3. Run
python main.py
```
